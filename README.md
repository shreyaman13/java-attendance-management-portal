# Attendance Management Portal

A Java-based Attendance Management System developed using Java Swing and CSV-based storage.

## Features

- Student attendance tracking
- GUI-based interface using Java Swing
- CSV file handling for data storage
- Attendance report management
- Student data management

## Technologies Used

- Java
- Java Swing
- CSV File Handling
- IntelliJ IDEA

## Project Structure

- `src/` → Source code
- `data/` → CSV data files
- `README.md` → Project documentation

## Learning Outcomes

Through this project, I improved my understanding of:

- GUI development in Java
- File handling using CSV
- Project structuring
- Object-oriented programming concepts
